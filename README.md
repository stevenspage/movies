# 2025年美剧推荐

一份精心整理的2025年即将播出的美剧列表。

## 美剧列表

### A Thousand Blows
*   **评分:** 78/100
*   **简介:** 一部由《浴血黑帮》主创史蒂文·奈特打造的历史剧。该剧背景设定在19世纪80年代的伦敦东区，探索了非法地下拳击的世界。

### Acapulco (Season 4)
*   **评分:** 待定
*   **简介:** 在第四季也是最后一季中，故事将继续其双线叙事。在现代，年长的马克西莫（欧赫尼奥·德尔维斯饰）正致力于让拉斯科利纳斯度假村重现昔日辉煌，准备盛大重新开业。与此同时，在1986年，年轻的马克西莫（恩里克·阿里宗饰）将面临新的挑战，因为一家竞争对手的酒店威胁到了拉斯科利纳斯的顶级地位，迫使他必须不惜一切代价来确保度假村的未来。

### Alien: The Series
*   **评分:** 待定
*   **简介:** 一部科幻恐怖剧集，由诺亚·霍利创作，是1979年电影《异形》的前传。故事背景设定在2120年，即原版电影事件发生的两年前。

### American Primeval
*   **评分:** 8.0/10
*   **简介:** 一部六集的西部迷你剧，于2025年1月9日在Netflix首播。该剧背景设定在1857年的犹他领地，探讨了美国西部因土地和资源而引发的暴力冲突。

### Andor (Season 2)
*   **评分:** 待定
*   **简介:** 备受赞誉的《星球大战》系列剧集《安多》的第二季也是最后一季，将于2025年4月22日在Disney+首播。该季将由12集组成，分四周以三集一批的方式发布。

### Billy Joel: And So It Goes
*   **评分:** 待定
*   **简介:** 一部名为《比利·乔尔：就这样》的全新两集HBO纪录片，将于2025年7月首播。

### Black Rabbit

*   **简介:** 一部即将上映的限定剧，计划于2025年9月18日在Netflix首播。该剧由裘德·洛和杰森·贝特曼主演，饰演一对兄弟。

### Chad Powers
*   **评分:** 待定

### Chief of War

### Creature Commandos

*   **评分:** 待定

### Department Q

### Dexter: Resurrection
